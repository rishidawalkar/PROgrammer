Operators.


  Java provides many types of operators which can be used according to the need. They are classified based on the functionality they provide.
  Some of the types are-

   1.Assignment operator
   Assignment operators are used in Java to assign values to variables. For example,

   int age;
   age=5;

  2. Arithematic operators
  Arithmetic operators are used to perform mathematical operations like addition, subtraction, multiplication, etc.



  3.Unary operators

  The unary operator performs operations on only one operand.
  some of the unary operators are: + , - , ++ , -- , ! .

  4.Equality and Relational opertors
  The equality and relational operators determine the relationship between the two operands. It checks if an operand is greater than, less than,
  equal to, not equal to and so on. Depending on the relationship, it is evaluated to either true or false.



  4.instanceof oparator
  In addition to relational operators, there is also a type comparison operator 'instanceof' which compares an object to a specified type.

  5.Logical oparator
  The logical operators || (conditional-OR) and && (conditional-AND) operate on boolean expressions.

  6.Ternary operator
  The conditional operator or ternary operator ?: is shorthand for the if-then-else statement.The syntax of the conditional operator is:

   variable = Expression ? expression1 : expression2

  Here's how it works.
  If the Expression is true, expression1 is assigned to the variable.
  If the Expression is false, expression2 is assigned to the variable.

  7.Bitwise and Bit shift operator
  To perform bitwise and bit shift operators in Java, these operators are used.
