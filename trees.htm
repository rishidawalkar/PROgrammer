Trees.


  A Tree structure means that the data is oraganized in a hierarchical manner.
  Tree is one of the most powerful and advanced data structures. It is a non-linear data structure compared to arrays, linked lists, stack and queue.
  It represents the nodes connected by edges.

                                                                                                              

  As we can refer from the figure that tree has Root node, parent node, child node and the leaf node.
  Root node is parent node of all the nodes of the tree as it is the uppermost node.
  Parent node is an immediate predecessor of a node.
  Child nodes are all immediate successors of a node.
  Siblings are the nodes which are at the same level of the tree.
  And, Leaf node is a node which do not have its child nodes.
